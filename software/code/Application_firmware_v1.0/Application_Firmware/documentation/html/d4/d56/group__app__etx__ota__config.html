Detailed Description

This file should contain all the Application ETX OTA Protocol Configurations made by the user/implementor.

Note
It is highly suggested not to directly edit the Configurations Settings defined in the Default ETX OTA Protocol Configuration File file. Instead of doing that whenever requiring different Configuration Settings, it is suggested to do that instead in this additional header file named as "app_etx_ota_config.h". However, to enable the use of this additional header file, you will also need to change the ENABLE_APP_ETX_OTA_CONFIG define, that is located at Default ETX OTA Protocol Configuration File, to a value of 1.
