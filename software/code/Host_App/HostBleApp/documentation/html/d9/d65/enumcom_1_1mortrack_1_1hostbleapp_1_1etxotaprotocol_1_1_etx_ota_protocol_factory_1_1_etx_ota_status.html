Detailed Description

ETX OTA Exception code definitions.

These Exception Codes are returned by some of the methods of the Classes that use the com.mortrack.hostbleapp.etxotaprotocol.EtxOtaProtocolFactory.EtxOtaStatus Interface with the purpose of indicating resulting status of having executed their respective processes. For example, to indicate that the process executed by a certain method was successful or that it has failed.
