Detailed Description

This Interface provides several template methods for the Concrete Products com.mortrack.hostbleapp.etxotaprotocol.UartEtxOtaProtocol and com.mortrack.hostbleapp.etxotaprotocol.BleEtxOtaProtocol that are within the com.mortrack.hostbleapp.etxotaprotocol.EtxOtaProtocolFactory Factory Class.

The provided methods from this Interface should enable the implementer to initialize and send Payload Data to slave devices via the ETX OTA Communication Protocol over the already chosen Hardware Protocol (i.e., either UART or BLE).

For more details on the available Type values for this field, see com.mortrack.hostbleapp.etxotaprotocol.EtxOtaProtocolFactory#EtxOtaPayloadType ).
