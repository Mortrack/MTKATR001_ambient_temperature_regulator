Detailed Description

ETX OTA Exception codes.

These Exception Codes are returned by the methods of the UartEtxOtaProtocol Class to indicate the resulting status of having executed the process contained in each of those methods. For example, to indicate that the process executed by a certain method was successful or that it has failed.
