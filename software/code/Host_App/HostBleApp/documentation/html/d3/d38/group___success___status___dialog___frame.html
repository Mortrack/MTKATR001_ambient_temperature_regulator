Detailed Description

This module contains the methods and event handlers required to generate and show to the user a Success Status Dialog Frame as part of the UI so that this UI can show Success Messages to the user.

The following gives a friendly code example of how to use com.mortrack.hostbleapp.jframes.SuccessStatusDialogFrame Class, which is easily achieved with a single code of line:

new SuccessStatusDialogFrame("The requested Payload has been successfully sent to the slave device.", this); // Call this from wherever you want from the MainFrame Class.
