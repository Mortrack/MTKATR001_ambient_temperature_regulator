void com.mortrack.hostbleapp.jframes.MainFrame.jButtonSendEtxOtaPayloadActionPerformed ( java.awt.event.ActionEvent  evt)
private

Event Handler method for whenever there is an Action Performed on the com.mortrack.hostbleapp.jframes.MainFrame.jButtonSendEtxOtaPayload Button.

Whenever this Event Handler method is called, the remaining data to be validated from the fields/items of the UI will be validated (some are validated the moment their values change). Then, if everything goes well, then the requested Payload will be sent to the chosen slave device via the ETX OTA Communication Protocol, the selected Hardware Protocol and also under the configurations defined in the UI by the user.

A J Dialog Form will be shown to inform the user of the result of having started an ETX OTA Transaction.

◆ dongleConfiguratorRs232ComportNumber

short com.mortrack.hostbleapp.jframes.MainFrame.dongleConfiguratorRs232ComportNumber
private

Current ETX OTA RS232 Comport chosen by the user for the Dongle Configurator API.

The valid and supported Comports are COM1, COM2, ..., COM32 in Windows, which stand for Comport Numbers 1, 2, ..., 32 respectively.

For Linux based Operative Systems, valid Comports are ttyS0, ttyS1, ..., ttyS15 with the Comport Numbers 1, 2, ..., 16 respectively. Other valid Comports for these O.S. are also ttyUSB0, ttyUSB1, ..., ttyUSB5 with the Comport Numbers 17, 18, ..., 22. Other valid Comports are also ttyAMA0, ttyAMA1, ttyACM0, ttyACM1, rfcomm0, rfcomm1, ircomm0, ircomm1, cuau0 and cuau1 with the Comport Numbers 23, 24, 25, 26, 27, 28, 29, 30, 31 and 32 respectively.

◆ dongleConfiguratorRs232ModeParity

char com.mortrack.hostbleapp.jframes.MainFrame.dongleConfiguratorRs232ModeParity = 'N'
private

Chosen Parity string with which we want the host to run the RS232 protocol for the Dongle Configurator API.

The valid parity values in the Teuniz RS232 Library that are used in the PcTool API are:

  • 'N' = None (i.e., No parity bit sent at all).
  • 'O' = Odd (i.e., Set to mark if the number of 1s in data bits is odd.
  • 'E' = Even (i.e., Set to space if the number of 1s in data bits is even.
