Detailed Description

This module contains the methods and event handlers required to generate and show to the user an Error Status Dialog Frame as part of the UI so that this UI can show Error Messages to the user.

The following gives a friendly code example of how to use com.mortrack.hostbleapp.jframes.ErrorStatusDialogFrame Class, which is easily achieved with a single code of line:

new ErrorStatusDialogFrame("", "The given Payload File Path does not exist.", this); // Call this from wherever you want from the MainFrame Class.

Another friendly code example is the following:

int codeNumber = 17;
new ErrorStatusDialogFrame("ETX OTA Exception Code = " + codeNumber, "The requested Payload could not be sent to the slave device.", this); // Call this from wherever you want from the MainFrame Class.
