Detailed Description

ETX OTA Header Type Packet's parameters structure.

This structure contains all the fields of an ETX OTA Packet of ETX_OTA_STATE_HEADER Type.

The following illustrates the ETX OTA format of this Type of Packets:

SOF Packet
Type 		Len Header
Data 		CRC EOF
1B 1B 2B 16B 4B 1B

where B = byte(s).
