Detailed Description

Header Data parameters structure.

This structure contains all the fields of the Header data that is expected to be received by our external device (connected to to it via Command_Line_Arguments::COMPORT_NUMBER ) in a ETX_OTA_PACKET_TYPE_HEADER Type Packet (i.e., in a Header Type Packet).
