This file should contain all the Dongle Configurator Configurations that are wished to be changed from their default values, which are defined as such in the Dongle Configurator module file.

Note
It is highly suggested not to directly edit the Configurations Settings defined in the Dongle Configurator Configuration File. file. Instead of doing that whenever requiring different Configuration Settings, it is suggested to do that instead in this additional header file named as "app_dongle_configurator_config.h". However, to enable the use of this additional header file, you will also need to change the ENABLE_APP_DONGLE_CONFIGURATOR_CONFIG define, that is located at Dongle Configurator Configuration File., to a value of 1.
For more details on how to use this Application Dongle Configurator Configuration File , see the Readme.md file of the project on which the Dongle Configurator library is being used at.
