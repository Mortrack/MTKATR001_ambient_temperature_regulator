Detailed Description

This module provides the functions required to enable the application to be able to connect and configure an HM-10 BT Device as Central Mode via the HM-10 Bluetooth module (for Windows and Linux) .

◆ Command_Line_Arguments

enum Command_Line_Arguments

Command Line Arguments definitions.

These definitions are used as identifiers for the Command Line Arguments that should be received right after the resulting compiled file from this Dongle Configurator module is executed via the terminal window.

Note
For more details see main and the Readme.md file of the project that uses the Dongle Configurator module .
Enumerator
TERMINAL_WINDOW_EXECUTION_COMMAND 

Command Line Argument Index 0, which should contain the string of the literal terminal window command used by the user to execute the Dongle Configurator module program.

COMPORT_NUMBER 

Command Line Argument Index 1, which should contain the Comport with which the user wants the Dongle Configurator module program to establish a connection with via RS232 protocol.

RS232_BAUDRATE 

Command Line Argument Index 2, which should contain the Baud rate that wants to be used on the Serial Port (i.e, Command_Line_Arguments::COMPORT_NUMBER ) with which a communication with the HM-10 BT Device will be made.

Note
For valid values, see Teuniz RS232 Library .
RS232_MODE_DATA_BITS 

Command Line Argument Index 3, which should contain the Data-bits with which we want the host to run the RS232 protocol.

Note
For valid values, see Teuniz RS232 Library .
RS232_MODE_PARITY 

Command Line Argument Index 4, which should contain the Parity with which we want the host to run the RS232 protocol.

Note
For valid values, see Teuniz RS232 Library .
RS232_MODE_STOPBITS 

Command Line Argument Index 5, which should contain the Stop-bits with which we want the host to run the RS232 protocol.

Note
For valid values, see Teuniz RS232 Library .
RS232_IS_FLOW_CONTROL 

Command Line Argument Index 6, which should contain the Flag value to indicate whether we want the host to run the RS232 protocol with Flow Control with a "1" or otherwise with a "0" .

Note
For valid values, see Teuniz RS232 Library .
SEND_PACKET_BYTES_DELAY 

Command Line Argument Index 7, which should contain the desired delay in microseconds to be requested after having send a byte to the HM-10 BT Device via the HM-10 Bluetooth module (for Windows and Linux) .

TEUNIZ_LIB_POLL_COMPORT_DELAY 

Command Line Argument Index 8, which should contain the desired delay in microseconds that is to be requested to apply each time before calling the get_hm10_ota_data function of the HM-10 Bluetooth module (for Windows and Linux) .

enum DONGLE_CONFIGURATOR_Status

Dongle Configurator Exception codes.

These Exception Codes are returned by the functions of the Dongle Configurator module module to indicate the resulting status of having executed the process contained in each of those functions. For example, to indicate that the process executed by a certain function was successful or that it has failed.

Enumerator
DONGLE_CONFIGURATOR_EC_OK 

Dongle Configurator Process was successful.

DONGLE_CONFIGURATOR_EC_STOP 

Dongle Configurator Process has been stopped.

DONGLE_CONFIGURATOR_EC_NR 

Dongle Configurator Process has concluded with no response from Host.

DONGLE_CONFIGURATOR_EC_NA 

Dongle Configurator Process Not Applicable.

DONGLE_CONFIGURATOR_EC_ERR

static DONGLE_CONFIGURATOR_Status send_hm10_central_mode_cmds ( DONGLE_CONFIGURATOR_API_t *  p_dongle_configurator_api)
static

Sends the corresponding AT Commands to the HM-10 Device to first initialize the HM-10 Library and to then configure the HM-10 Device so that it can be used as a Bluetooth Dongle Device.

Note
The RS-232 Serial Protocol to be used must be opened independently to this function via the RS232_OpenComport function from the Teuniz Library.

Local variable teuniz_rs232_lib_comport: Should hold the converted value of the actual comport that was requested by the user but into its equivalent for the Teuniz RS232 Library (For more details, see the Table from Teuniz RS232 Library ).
