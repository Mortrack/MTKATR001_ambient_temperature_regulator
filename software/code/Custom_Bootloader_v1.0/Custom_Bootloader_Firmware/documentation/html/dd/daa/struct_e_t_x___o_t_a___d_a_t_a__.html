Detailed Description

ETX OTA Data Type Packet's parameters structure.

This structure contains all the fields of an ETX OTA Packet of ETX_OTA_STATE_DATA Type.

The following illustrates the ETX OTA format of this Type of Packets:

SOF Packet
Type 		Len Data CRC EOF
1B 1B 2B data_len B 4B 1B

where B = byte(s).

Note
The CRC is not within the fields of this structure because this structure stands for one of several Data Chunks to be received. Therefore, the strategy that has been applied in the Firmware Update module module is to append all the received Data Chunks to then calculate their CRC and validate with the one received for that (i.e., for the entire appended Data Chunks), which is the package_crc field from the header_data_t structure.
Because of the same reason, the EOF is also not within the fields of this structure but it is always validated at the end of each Data Chunk received.
