Detailed Description

ETX OTA Response Type Packet's parameters structure.

This structure contains all the fields of an ETX OTA Packet of ETX_OTA_PACKET_TYPE_RESPONSE Type.

The following illustrates the ETX OTA format of this Type of Packets:

SOF Packet
Type 		Len Status CRC EOF
1B 1B 2B 1B 4B 1B

where B = byte(s).
